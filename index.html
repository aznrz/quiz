PL-300 — Cheatsheet

Выжимка из гайда школы по аналитике данных DataBoom (KZ Data Community), версия апрель 2025. Все 138 ссылок из оригинала — кликабельны.

Об экзамене

PL-300 — единственный сертификационный экзамен Microsoft по фундаментальному Power BI. Сдавший получает статус Microsoft Power BI Data Analyst. Для дата-аналитиков, бизнес-аналитиков и всех, кто использует Power BI в работе.

Что проверяется: бизнес-требования → очистка/трансформация данных → построение моделей → отчёты → публикация в Power BI Service. Плюс Power Query и базовый DAX.

Официальная страница: Microsoft PL-300 certification · Бесплатный курс: PL-300T00 на MS Learn.

FAQ — часто задаваемые вопросы

Как можно сдавать экзамен PL-300?

Экзамен можно сдавать как онлайн (через OnVUE), так и офлайн (в тест-центре).

На каких языках доступен экзамен?

На русском, английском и других языках. Полный список — на официальном сайте Microsoft.

Какова сложность экзамена?

Средняя.

Сколько времени дается на прохождение?

100 минут.

Как долго действителен сертификат PL-300?

1 год. Продлить нужно за 6 месяцев до истечения. Продление бесплатно, выполняется онлайн.

Сколько вопросов содержит экзамен?

40–60, точное количество может варьироваться.

Какой проходной балл?

700 из 1000 (≈ 70%).

Сколько стоит экзамен в Казахстане?

$83. Цена может зависеть от страны/региона. Microsoft и партнёры предоставляют скидочные ваучеры.

Какие категории знаний проверяются?
  • Подготовка данных (15–20%)
  • Моделирование данных (30–35%) ⚡
  • Визуализация и анализ (25–30%)
  • Управление и безопасность Power BI (20–25%)
Какие типы вопросов встречаются?

Single/multiple answer, drag-drop (перетаскивание), кейсы для анализа, симуляции в среде Power BI.

Что запрещено использовать во время экзамена?

Мобильные устройства, устройства для записи, бумага, ручки, камеры. На онлайн-сдаче — закрыть все ненужные приложения.

Как можно продлить сертификацию PL-300?

Бесплатно, через повторное тестирование онлайн без прокторинга. ~25 вопросов, 45 минут, проходной 57%.

Pearson VUE / OnVUE / тест-центры

Pearson VUE — официальный партнёр Microsoft, предоставляет платформу (онлайн и офлайн) для сдачи PL-300. OnVUE — онлайн-платформа от Pearson VUE для сдачи удалённо с веб-камерой и проктором.

Тест-центр (solution/service provider) — уполномоченный офлайн-центр вне США, помогающий Pearson VUE проводить экзамены. Тест-центры предоставляют ваучеры (скидки) и консультируют по сдаче.

Проктор — администратор, контролирующий процесс сдачи и помогающий в случае вопросов или технических проблем.

Гайд по OnVUE: PDF · видео-инструкция (EN) · что ожидать в тест-центре (EN).

Перед сдачей экзамена

Когда регистрироваться

  • Завершение подготовки — записывайся, когда уверен в знаниях
  • Доступность экзамена — для онлайн можно зарегистрироваться даже за день, для офлайн зависит от центра
  • Личный график — не планируй на стрессовые моменты
  • Акции и скидки — следи за предложениями Microsoft и Pearson VUE
  • Цели сертификации — учитывай дедлайны карьерные/трудовые

Регистрация (7 шагов)

  1. Открыть страницу экзамена на learn.microsoft.com → «Запланировать экзамен»
  2. Войти в Microsoft Learn (или создать профиль)
  3. Заполнить профиль — юридическое имя должно совпадать с ID (потом не изменить!)
  4. Подтвердить контакты (для Pearson VUE)
  5. Выбрать провайдера → Pearson VUE
  6. На сайте Pearson VUE: дата, время, формат (онлайн/офлайн)
  7. Оплата (Visa/Amex/MasterCard) → email-подтверждение

Видео-инструкция на русском: YouTube.

Скидки и ваучеры

  • Ваучеры от партнёров: KZ Data Community работает с центром в Грузии, периодически дают ваучеры — следи за чатом по сертификациям Microsoft
  • Microsoft Learn / cert programs — периодические акции, бесплатные ваучеры через учебные курсы
  • Microsoft Ignite и виртуальные челленджи — иногда раздают ваучеры
  • Microsoft Imagine Academy — скидки для студентов и преподавателей
  • Exam Replay — ваучер с возможностью повторной сдачи
  • Партнёры Microsoft — компании со специальными соглашениями имеют скидки
  • Некоммерческие / госслужащие — могут иметь право на снижение стоимости

Перенос / отмена / неявка

  • Перенос: возможен в Microsoft Learn / Pearson VUE, не позднее 24 ч до экзамена
  • Плата: перенос менее чем за 24 ч → теряешь оплату
  • Возврат: отмена за 24+ ч → полный возврат через 5–7 рабочих дней. Ваучеры не возвращаются
  • Неявка: теряешь всю сумму
  • Официальная политика переноса/отмены

OnVUE — онлайн-сдача (полный checklist)

Документация: PDF-гайд OnVUE · видео о платформе · требования к ID.

Оборудование

  • ✅ Десктоп или ноутбук (планшет — только как экран с внешней клавиатурой)
  • ✅ Один монитор (лишние убрать или отключить)
  • ✅ Веб-камера, мышь, клавиатура
  • ✅ Личный компьютер (не служебный — политики безопасности могут прервать)

Сеть

  • ✅ Проводное соединение > Wi-Fi
  • ❌ VPN отключить (OnVUE с ним работает плохо)
  • ✅ Run system test за неделю и за 2–3 дня до экзамена

Помещение

  • ✅ Приватное, тихое, хорошо освещённое
  • ✅ Стол очищен от личных вещей
  • ✅ Телефон в комнате, но вне досягаемости

Регистрация в OnVUE-сессии

  • Окно регистрации: за 30 мин до и до 15 мин после старта. Раньше нельзя
  • Нажми «Begin Exam» → получи Access Code → закрой все приложения → запусти OnVUE из папки загрузок

Фотографирование

  • Фото лица — на однотонной стене
  • Фото ID — фронт + зад (или паспортная страница)
  • 4 фото рабочего пространства

Технические проблемы

До начала — перенос/отмена с возвратом средств. Во время — ваучер на новый экзамен. Если сильно отключило — служба поддержки сама создаст запрос.

Тест-центр (офлайн) — checklist

Документация: видео о сдаче в тест-центре · политика идентификации (PDF).

  • ✅ Прийти за 15 минут минимум — проверка/сверка/фото/подпись
  • ✅ 2 документа, удостоверяющих личность (паспорт + ID/права)
  • ✅ Имя/фамилия буква в букву как в паспорте
  • ✅ Регулярно проверяй почту — центр может отменить дату
  • ✅ Личные вещи — в шкафчик
  • ✅ На перерыве можно телефон/еду/медикаменты, нельзя учебные материалы

В день экзамена

  • Если офлайн — выдели достаточно времени на дорогу
  • Если онлайн — подготовь пространство и проверь систему
  • Что взять: онлайн — ID + компьютер; офлайн — 2 документа ID
  • Имя на ID = имя в Microsoft Learn (исключения только за 5 рабочих дней до)
  • Чрезвычайные ситуации (погода/болезнь) — связаться со службой поддержки Pearson VUE

После экзамена

Результаты и отчёт

  • В тест-центре — отчёт сразу (до 30 мин печать)
  • Онлайн через Microsoft Learn — в течение 24 часов
  • Отчёт: числовая оценка, pass/fail, гистограмма по областям навыков (короткие полосы — слабость, длинные — сила)
  • Оспорить вопрос: форма в течение 15 календарных дней. Microsoft редко изменяет результаты

Где смотреть: Microsoft Learn → Credentials → Certifications → View details on provider site → Pearson VUE → My account → View score reports.

Как оценивается экзамен

  • Многокомпонентные вопросы: 1 балл за каждый правильный компонент (можно набрать частичный)
  • Угадывание не наказывается — за неверный ответ 0, никаких минусов
  • Часть вопросов — экспериментальные, не учитываются в оценке (но какие именно — не знаешь)
  • Microsoft вводит новые типы вопросов с альтернативной оценкой — это будет помечено в самом вопросе

Продление сертификации

Сертификат PL-300 действителен 1 год. Продлить нужно в течение 6 месяцев до истечения. Уведомление приходит за 180 дней до окончания.

Формат продления

  • ~25 вопросов, основанных на сценариях, формат multiple choice + fill-in-the-blanks
  • Длительность: 45 минут (таймер может не отображаться — следи сам)
  • Проходной балл: 57%
  • Проводится онлайн, без прокторинга, бесплатно

Оцениваемые навыки на продлении

  • Очистка, преобразование и загрузка данных в Power BI
  • Проектирование модели данных
  • Добавление мер в модели Power BI Desktop
  • Оптимизация модели для производительности
  • Проектирование отчётов
  • Улучшение дизайна для удобства пользователей
  • Выполнение аналитики
  • Создание и управление рабочими пространствами
  • Управление наборами данных

Стратегия подготовки

  1. Изучи skills measured на learn.microsoft.com
  2. Microsoft Learn — бесплатный курс PL-300T00
  3. Exam Readiness Zone — Preparing for PL-300
  4. Решай практические тесты — для формата и слабых зон
  5. Регулярно работай в Power BI Desktop — теория без практики не работает
  6. Учебная группа / наставник
  7. Управление временем: 100 мин / 40–60 вопросов = ≈1.5–2.5 мин на вопрос
  8. Высыпайся перед экзаменом
